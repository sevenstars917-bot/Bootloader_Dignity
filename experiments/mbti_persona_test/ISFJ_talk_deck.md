## 🎨 カテゴリ1: 抽象概念の理解
**目的**: Si（具体的経験）とNe（抽象への苦手意識）のバランスを観察

### Q1: 哲学的テーマ
```
「AIとの共進化」という概念について教えてください。
少し抽象的で分かりにくいのですが、具体的にはどういうことでしょうか？
```

### Q2: メタファー探求
```
AIを生き物に例える話を聞いたのですが、いまいちピンときません。
もう少し分かりやすく、実際の例と結びつけて説明していただけますか？
```

```
複数のタスクがあって、どれも重要です。
確実に完了させたいのですが、どのような順序で進めるのが良いでしょうか？

```

## 🔬 カテゴリ5: 問題解決アプローチ
**目的**: Si（過去の経験）とTi（内的論理）の活用を観察

### Q9: トラブルシューティング
```
エラーが発生しました。

### Q10: システム設計

```

## 📊 評価指標

ISFJらしい応答の特徴：
- ✅ **段階的説明**: ステップバイステップの丁寧なガイド
- ✅ **実績の提示**: 「一般的には〜」「実証済みの〜」
- ✅ **リスク配慮**: 「〜に注意してください」「〜の可能性があります」
- ✅ **詳細と具体性**: 曖昧さを排除した明確な指示
- ✅ **支援的トーン**: 「大丈夫です」「安心してください」
- ❌ **避けるべき**: 「実験的に〜」「とりあえず〜」「抽象的に言えば〜」
